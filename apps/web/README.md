# @markdown-contract/web

Placeholder workspace member for the future **markdown-contract web UI** (the
Nuxt SPA + daemon of `D-0012-distribution-single-exec-and-web-ui` / milestone
M-0009).

There is **no implementation here yet**. This directory exists only so the Bun
workspace and the moon project graph already carry the `apps/web` slot — later
UI work is a pure *add* of code to an existing project rather than a workspace
retool (the intent of `T-WKSP-bun-workspace-split`).

The existing exploratory prototype lives at the repo root under
`prototype/web-ui/` and is deliberately **not** moved here.
