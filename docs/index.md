## Conventions

| Convention | Meaning |
| ---------- | ------- |
| **Wikilinks** `[[name]]` | Resolve by filename match. Catalogued entities use `<prefix><N>-<slug>`; milestones use `[[<id>]]` (random base-36, e.g. `[[M-ML68]]`; legacy `M0000`/`M0001` grandfathered) |
| **Transclusion** `![[file#^summary]]` | Embeds an entity's summary block — how the summary cells stay in sync with their source entities |
| **Markdown links** `[name](path)` | Used for ADR folders and directories where wikilinks would be awkward |
