# References

> External documentation, research, and artifacts SDLC leans on,
> catalogued once. **Generated** from the reference entities under
> [`./planning/references/`](./planning/references/) by the `docs`
> skill (`sdlc docs generate references`). **Do not hand-edit** — edit
> the reference entity (or `sdlc reference create` a new one) and
> regenerate. Summary cells transclude each reference's `^summary`
> block.
